# homebrew-tap

Personal [Homebrew](https://brew.sh) tap for **Ratune**, a Rust-based TUI music player for Subsonic-compatible servers.

Upstream project: [github.com/acmagn/ratune](https://github.com/acmagn/ratune)

## How do I install these formulae?

To install Ratune, run:

```bash
brew tap acmagn/tap
brew install ratune
```

macOS only: the formula installs prebuilt **Apple Silicon** or **Intel** binaries from GitHub Releases, depending on your Mac.

## Maintenance

This repository is updated whenever a new release of Ratune is published.

## Troubleshooting

If you run into issues with the formula or the tap, please open an issue in the **[main Ratune repository](https://github.com/acmagn/ratune/issues)**.
